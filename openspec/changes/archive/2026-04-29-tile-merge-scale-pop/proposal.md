## Why

合并发生时现有反馈偏粒子与音效，方块本体缺少「撞击成型」的肉感；在幸存合并方块上做**短时放大再复原**，可强化合并可读性且不改变游戏规则。

## What Changes

- 在一次滑动结算后，凡产生合并，对应**合并位置上的幸存方块**（显示合并后数值的那一格）播放一次**缩放脉冲**：相对静止尺度略微放大后再平滑回到 **scale(1)**。
- 动画为短时（例如约 150～350ms 量级，以实现为准）、可与既有粒子并行；不改变格子占位与触控命中语义（视觉上缩放应在格内完成或通过 `transform` 不破坏布局）。

## Capabilities

### New Capabilities

（无）

### Modified Capabilities

- **`h5-2048-game`**：增补「合并方块缩放反馈」规范性描述（幸存方块上的短时放大复原）；与既有「方块滑动与静止外观一致」兼容——缩放 SHALL 为短时变换且不替换整块投影皮肤。

## Impact

- **前端**：`Tile.vue`（缩放动画 class / CSS）、`Board.vue` 或 `useGame2048.ts`（合并幸存 tile id 或格坐标上一帧脉冲标记）。
- **依赖**：无新增运行时依赖。
- **静态部署**：纯 CSS 动画与 Vue 状态，不改变 itch.io 打包约束。
