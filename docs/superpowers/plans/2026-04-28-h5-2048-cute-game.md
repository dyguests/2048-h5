# 可爱风 2048 H5（Vue 3 + Vite）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver a mobile-friendly 2048 game as a Vite + Vue 3 + TypeScript SPA with cute styling, slide animations, sparkle particles, motion trails, keyboard + touch input, and itch.io-ready static `dist/` output (`base: './'`).

**Architecture:** Pure game logic in `src/game/*.ts` (Vitest). Vue composables orchestrate moves, input lock, and animation completion. DOM-based board and tiles with CSS `transform` transitions; one overlay `FxCanvas` handles particles and trails via Canvas2D + object pool + rAF.

**Tech Stack:** Vite 6, Vue 3.5, TypeScript 5.6, Vitest.

**Normative references:** `docs/superpowers/specs/2026-04-28-h5-2048-vite-vue-design.md`, `openspec/changes/h5-2048-cute-game/specs/h5-2048-game/spec.md`.

- [ ] **Step 2: Implement**

```ts
export function spawnTile(
  grid: Grid,
  rand: () => number = Math.random,
  nextId: () => string = () => crypto.randomUUID(),
): Grid {
  const empty: { r: number; c: number }[] = [];
  for (let r = 0; r < 4; r++) {
    for (let c = 0; c < 4; c++) {
      if (grid[r][c] === null) empty.push({ r, c });
    }
  }
  if (empty.length === 0) return grid;
  const pick = empty[Math.floor(rand() * empty.length)]!;
  const next = cloneGrid(grid);
  next[pick.r][pick.c] = { id: nextId(), value: rand() < PROB_TWO ? 2 : 4 };
  return next;
}

export function canMove(grid: Grid): boolean {
  for (let r = 0; r < 4; r++) {
    for (let c = 0; c < 4; c++) {
      if (grid[r][c] === null) return true;
      const v = grid[r][c]!.value;
      if (c < 3 && grid[r][c + 1]?.value === v) return true;
      if (r < 3 && grid[r + 1][c]?.value === v) return true;
    }
  }
  return false;
}

export function isGameOver(grid: Grid): boolean {
  return !canMove(grid);
}
```
